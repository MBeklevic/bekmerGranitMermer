# Project Brief

## Project
Static marble workshop website in Turkish for customer viewing only.

## Goals
- Present company, services, references, and contact info with a premium look.
- Provide a “Teklif Al” page to capture user intent (static-only, no backend).
- Keep all data static; no CMS, no auth, no forms posting.

## Scope
- Framework: Next.js (static site).
- Styling: Tailwind CSS, matching provided HTML designs.
- Pages:
  - Anasayfa (Home)
  - Hakkımızda (About)
  - Hizmetler (Services)
  - İletişim (Contact)
  - Referanslar (References)
  - Teklif Al (Request a Quote)
- Turkish localization for all content.

## Success Criteria
- Visual parity with provided designs.
- Responsive, fast, and accessible.
- Simple codebase maintainable without a backend.
